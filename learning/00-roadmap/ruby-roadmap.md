# Ruby Roadmap

Rails 业务开发者进阶 — **全免费公开资料**。

See [ruby-engineering-track.md](./ruby-engineering-track.md) for goals and time budget.

## Overview

```
RubyMonk Ascent（block / self / 闭包）
        ↓
Metaprogramming 工作坊 + Koans（对象模型 / define_method）
        ↓
拆 Rails 魔法（has_many / validates / 读 AR 源码）
        ↓
Sandi Metz OO 设计（重构自己的代码）
```

## Phase map

| Phase | Task | Duration | Key materials |
|-------|------|----------|---------------|
| R1 | rubymonk_ascent | 2 weeks | RubyMonk Primer Ascent + MP Ruby 入门 |
| R2 | metaprogramming_workshop | 2 weeks | Paolo Tokyo repo + koans + challenges |
| R3 | rails_internals | 2 weeks | RubyEvents 演讲 + `bundle open activerecord` |
| R4 | oo_design | 2 weeks | Sandi Metz 3 演讲 + 重构实战 |
